There is no personal or non-personal information collected directly by PerrotteWx.com. See the "Google Analytics" section for information regarding that service.

Google Analytics

This website uses a service called Google Analytics, provided by Google Inc. This tool collects the following information (note that this is a non-exhaustive list):

  • Which website referred you (i.e., how you got to the website).
  • Which pages you view while on PerrotteWx.com and for how long you view them.
  • The date and time that you accessed the website.
  • The operating system that you are using.
  • Which web browser you are using.
  • The city and country from which you accessed PerrotteWx.com.

Usage of the Google Analytics Data

All data usage falls within the Google Analytics Terms of Service. I use this data to better understand how and when people come to my site and which pages they view. That way, I can focus my efforts on upgrading the most popular pages with new and useful features. Some aggregate data is viewed to better understand which web browsers are used the most. This allows me to optimize my website for the most popular web browsers, thereby producing the best experience for the largest number of people possible. Please rest assured that No personally identifiable information is collected by this service.
